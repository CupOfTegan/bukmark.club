Join the Club

If you're interested in getting your site added to the BUKMARK.CLUB, all you need to do is:

  1. Make sure you're adding a personal website.
  2. Make sure the site offers a curated collection of bookmarks and/or links to other websites at a dedicated URL (i.e. /bookmarks, /links).
  3. Head over to [our GitHub repository] and create a pull request (or an issue) to add your site.
  4. We will review you PR (or issue). If everything's alright, your site will be added to the directory.

Note: we reserve the explicit right _not_ to add submission to the directory if we deem them unsuitable in any way.
